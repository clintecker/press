# Brand assets

Source logo files for the press docs site and repo. The docs site derives its
favicon, nav mark, and social card from these; the README shows the lockup.

The icons are the pilcrow mark in three colorways, plus two `press.` lockups:

- `press-icon-ink.png` — the mark, vermilion on ink (favicon, light-mode nav)
- `press-icon-vermilion.png` — the mark, paper on vermilion (dark-mode nav)
- `press-icon-outline.png` — the mark, ink on paper
- `press-lockup.png` — the `press.` lockup on paper (light backgrounds)
- `press-lockup-reversed.png` — the lockup on ink (dark backgrounds, social card)
