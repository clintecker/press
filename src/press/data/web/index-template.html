How This Book Was Built

Every edition above is compiled from the same plain Markdown source by Pandoc and LuaLaTeX, in one command, on every push. No format is edited by hand, and nothing is stated in two places.

Each build must survive a gauntlet before it may call itself a book: structural checks on the source, an editorial audit for synthetic prose and forbidden punctuation, a jargon linter with a maintained watchlist, and a rendered-page inspection that verifies trim size, embedded fonts, expected text, and that every page contains ink. The artifacts here were attached by the same continuous integration that refused earlier, worse versions.
