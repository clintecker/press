# Source note

The seed list was informed by:

- the terms supplied directly for this skill;
- Archie Wood's `archiewood/claudeisms` repository;
- the existing `human-writing`, `writing-well`, `elements-of-style`, and `hemingway-prose` skills.

External list:

```text
https://raw.githubusercontent.com/archiewood/claudeisms/refs/heads/main/claudeisms.csv
```

The external CSV is not bundled because the repository did not expose a license when this package was created.

The repository describes a simple frequency comparison between one user's Claude Code transcripts and Stack Overflow comments. Its author notes several limits: the sample is personal and domain-skewed, the analysis was not independently checked, and inflected forms were not grouped. This package therefore uses that list only to propose terms for review.

A high frequency ratio does not make a word wrong. Terms such as `idempotent`, `retry`, `diff`, and `lint` can be exact and necessary. The skill judges each use by meaning, audience, evidence, and repetition.
